# 🚀 AI Code Reviewer

An AI-powered Code Review platform that performs static code analysis, detects code smells, measures complexity, generates documentation, and provides intelligent code improvement suggestions using AI.

---

## 📌 Features

### 🔍 Static Code Analysis
- Detect syntax and linting issues
- Identify errors, warnings, and informational messages
- Line-by-line issue reporting
- Severity classification

### 🤖 AI Code Review
- AI-generated code review
- Best practice recommendations
- Optimization suggestions
- Security recommendations
- Readability improvements

### 📊 Complexity Analysis
- Cyclomatic Complexity
- File Complexity
- Number of Functions
- Number of Classes
- Lines of Code
- Function Complexity Report

### 🧹 Code Smell Detection
- Long Files
- Long Lines
- Deep Nesting
- Too Many Parameters
- Maintainability Suggestions

### 📚 Automatic Documentation
- Function Documentation
- Class Documentation
- API Endpoint Documentation

### 📂 File Upload Support
- Upload source code files
- Automatic language detection
- Instant analysis

### 📜 Review History
- Save every review
- View previous reports
- Track improvements

### 🌗 Theme Support
- Light Theme
- Dark Theme
- Responsive UI

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/yourusername/AI-Code-Reviewer.git

cd AI-Code-Reviewer
```

---

## Backend Setup

```bash
cd backend

npm install

npm run dev
```

---

## Frontend Setup

```bash
cd frontend

npm install

npm run dev
```

---

# 🔐 Environment Variables

Create a `.env` file inside backend.

```env
PORT=5000

DATABASE_URL=your_postgresql_database_url

JWT_SECRET=your_jwt_secret

GEMINI_API_KEY=your_gemini_api_key
```
